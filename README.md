# homebrew-tilefeed

Homebrew tap for [tilefeed](https://github.com/muimsd/tilefeed) — a PostGIS vector tile pipeline with incremental MBTiles updates via LISTEN/NOTIFY.

## Install

```bash
brew tap muimsd/tilefeed
brew install tilefeed
```

## Update

```bash
brew update
brew upgrade tilefeed
```

## Uninstall

```bash
brew uninstall tilefeed
brew untap muimsd/tilefeed
```
